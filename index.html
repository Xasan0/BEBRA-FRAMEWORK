Card Examples

There is a simplest card ever so you can use it like from calling a class "card". A card is a flexible and extensible content container. It includes options for headers and footers, a wide variety of content, contextual background colors, and powerful display options.
<!DOCTYPE html> < html lang = "en" > < head > < meta charset = "utf-8" > < meta name = "viewport" content = "width=device-width, initial-scale=1" > < title > BEBRA demo </ title > </ head > < body > < div class = "card" > </ div >
< div class = "card-image" > The Source </ div > < div class = "card-title" > The Title </ div >
< div class = "card-desc" > The Description </ div > < body > </ html >
